Training Planner

**Автор:** [Анастасия Трифонова]

Описание
Приложение для планирования и учета тренировок. Позволяет добавлять записи о физической активности, сохранять их в локальный JSON-файл и фильтровать список по дате или типу упражнений.

Функционал
- Добавление тренировки (дата, тип, время).
- Валидация данных (проверка формата даты и положительного числа минут).
- Автоматическое сохранение в `trainings.json`.
- Живой поиск/фильтрация по списку.

Примеры использования
1. Введите дату `25.10.2023`.
2. Выберите тип `Силовая`.
3. Введите длительность `60`.
4. Нажмите "Добавить". Запись появится в таблице снизу.
5. В поле фильтра "Тип" введите "Кардио", чтобы увидеть только бег или велосипед.
